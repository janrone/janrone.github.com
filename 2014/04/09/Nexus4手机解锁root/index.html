Nexus4手机 解锁 root

发表于 2014-04-09   |   分类于 硬件   |     |   本文共被围观 次

解锁

1.打开手机调试功能，安装ADB，链接电脑。

2.shell中直接运行：adb reboot bootloader ，如图：

3. 接着运行： fastboot oem unlock

出现上图，按音量键盘，选择yes ，这时你手机上的所有数据都将被抹去,LOCK STATE将变成unlocked 。

ROOT

1. 进入fastboot模式, 可以用 adb reboot bootloader ，也可以在关机状态下，同时按音量+/-和电源键进入fastboot模式。

2.下载CF-Auto-root工具 nexus 4 安卓4.4.2 KOT49H root包下载

如果运行SuperSu 提示更新su二进制文件，需要重新root。

3. 解压并根据您的操作系统选择执行下面的文件

Windows:
— 双击运行root-windows.bat

  • Linux
    — chmod +x root-linux.sh
    — 运行 root-linux.sh

  • Mac OS X
    — chmod +x root-mac.sh
    — 运行 root-mac.sh
